## 下载方式：

选择想要的模型，以1.8为例

![](./..\readme_imgs\HY-MT下载1.png)

![](./..\readme_imgs/HY-MT下载2.png)

将下载的文件全部保存到“YT_AISpider/model/HY-MT1.5-1.8B”（即项目位置/model/模型名）文件夹下

![](./..\readme_imgs/HY-MT存放位置.png)
